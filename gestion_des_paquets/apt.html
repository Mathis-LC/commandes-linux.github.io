Gestionnaire de paquets APT présent sur les distributions basés sur Debian
Commande Utilité
apt update Met à jour la liste des paquets disponibles à partir des dépôts. Action indispensable avant toute installation
apt install Installe le paquet spécifié ainsi que toutes ses dépendances
apt remove Désinstalle le paquet, mais conserve ses fichiers de configuration (utile si l'on souhaite le réinstaller plus tard)
apt purge Désinstalle le paquet et supprime tous ses fichiers de configuration
apt autoremove Supprime les dépendances qui ont été installées automatiquement et ne sont plus utilisées par aucun paquet
apt search Recherche un paquet dans les dépôts locaux en fonction d'un mot-clé et affiche une description concise du paquet
