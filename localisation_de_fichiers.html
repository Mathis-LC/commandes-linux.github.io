Commande Utilité
cut Extrait des sections à partir de chaque ligne d'un fichier ou d'une entrée standard
find Recherche dans toute l'arborecence d'un système de fichiers à partir d'un point donné les fichiers correspondant à l'expression régulière passée en paramètre
grep Recherche dans les fichiers nommés dans la ligne de commande les lignes qui contiennent l'expression régulière spécifiée
locate Recherche une base de donées de fichiers système préinstallée pour retrouver rapidement un fichier
which Affiche le chemin complet d'accès à une commande
